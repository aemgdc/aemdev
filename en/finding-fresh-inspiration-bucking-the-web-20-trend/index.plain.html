Look at web design these days: Gradients, bursts, big type, rounded corners, pastel colors, and reflections rule the roost. This slavish adherence to Web 2.0 design trends by the “chorus” is resulting in a big snoozefest.

Now, I would have to agree with Ozon on one thing – definitely the originality of some web 2.0 sites is a bit lacking – especially as folks figure out how to use some of these features for the first time – like these little fade-in boxes on Digg or the drag-around popup layers in Bluedot or Otavo.
